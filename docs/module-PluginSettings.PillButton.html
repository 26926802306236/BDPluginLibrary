Class: PillButton

PluginSettings.PillButton(label, help, leftLabel, rightLabel, isRightSelected, callback, options)

Creates a PillButton where the left and right side have their own label. It is important to note that the checked property here follows the same standard as a normal Discord switch. That is to say if the value is true then right side was selected, if the value is false then the left side was selected.
